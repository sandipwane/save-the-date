# Save the Date - Sandip & Anokhi

![Wedding Invitation Preview](screenshot.png)

A mobile-friendly wedding invitation website for December 2025.

**Live Site:** [invite.sandipanokhi.space](https://invite.sandipanokhi.space)

## Features

- Hero image carousel with smooth transitions
- RSVP form with EmailJS integration
- Add to Calendar support (Google, Apple, Outlook, Yahoo)
- Confetti animation on RSVP
- Mobile-first responsive design

## Tech Stack

- Vanilla HTML/JavaScript
- Tailwind CSS (CDN)
- Lucide Icons
- GitHub Pages hosting
